About LiveCodes

LiveCodes is an open-source client-side code playground. It is packed with features and offers support for 80+ languages/frameworks. It can be used as a standalone app or can be embedded in any web page. It is very flexible with a wide range of configuration options. A powerful SDK is available to facilitate embedding and communication with embedded playgrounds.
