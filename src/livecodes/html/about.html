About LiveCodes

LiveCodes is an open-source, feature-rich, client-side code playground. Currently, 80+ languages/frameworks are supported. It can be used as a standalone app or can be embedded in any web page. There are many ways to prefill playgrounds with code.

A wide range of configuration options makes it very flexible. A powerful SDK (for JS/TS, React and Vue) facilitates embedding and communicating with playgrounds. Comprehensive documentations are available with code samples, live demos and screenshots.
